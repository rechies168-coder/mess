SMOS (System for Monitoring of glass-cutting Equipment) — автоматизированная система контроля технического состояния промышленного оборудования на предприятиях по обработке стекла. Обеспечивает непрерывный мониторинг температуры, нагрузки и производительности стеклорезных станков, плавильных печей и формовочных машин в реальном времени.

Требования
Перед установкой убедитесь, что у вас установлено:

Python 3.8 или выше

MongoDB 4.4 или выше (для хранения телеметрии)

Git (для клонирования репозитория)

Установка
bash
# 1. Клонируйте репозиторий
git clone https://github.com/your-org/SMOS.git
cd SMOS

# 2. Создайте виртуальное окружение
python -m venv .venv

# 3. Активируйте окружение
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Инициализируйте базу данных тестовыми данными
python scripts/initialize_mongodb.py
Конфигурация
Основные настройки задаются через переменные окружения. Скопируйте пример конфигурации и отредактируйте под себя:

bash
cp .env.example .env
Ключевые параметры:

Переменная	Описание	Пример
MONGODB_URI	Адрес сервера MongoDB	mongodb://localhost:27017
DB_NAME	Имя базы данных	smos_db
LOG_LEVEL	Уровень логирования	INFO
UPDATE_INTERVAL	Интервал опроса оборудования (сек)	10
Запуск
bash
# Запуск приложения
python src/gui/app.py
После запуска откроется окно входа. Используйте тестовые учетные данные:

Роль	Логин	Пароль
Администратор	admin	admin123
Оператор	operator	operator123
Наблюдатель	viewer	viewer123
Сборка и тестирование
Сборка исполняемого файла (опционально)
bash
# Установка PyInstaller
pip install pyinstaller

# Сборка
pyinstaller --onefile --windowed --name "SMOS" src/gui/app.py
Запуск тестов
bash
# Установка pytest
pip install pytest

# Запуск всех тестов
pytest tests/ -v

# Запуск с отчетом о покрытии
pytest tests/ --cov=src/smos --cov-report=html
